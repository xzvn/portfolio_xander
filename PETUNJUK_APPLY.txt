PATCH MODERNISASI HALAMAN ADMIN — SIDEBAR TIDAK DIUBAH

Perubahan:
- Background halaman admin memakai gradasi navy-indigo lembut.
- Topbar menjadi glass modern dengan status workspace aktif.
- Card tidak lagi putih polos; memakai permukaan bergradasi.
- Statistik, form, tabel, pesan, proyek, pengalaman, dan profil
  memiliki depth, border, shadow, serta hover profesional.
- Input dan select lebih interaktif saat hover dan focus.
- Card memiliki spotlight halus yang mengikuti pointer desktop.
- Konten masuk dengan animasi reveal ringan.
- Tombol memiliki ripple saat ditekan.
- Mobile tetap responsif dan efek berat otomatis dikurangi.
- Tampilan dan perilaku sidebar tetap seperti sebelumnya.

CARA MENERAPKAN

1. Ekstrak ZIP.
2. Salin seluruh isi patch ke:
   D:\PA ASDOS\portfolio_xander
3. Pilih Replace the files in the destination.
4. Restart Flask:
   Ctrl + C
   python app.py
5. Buka halaman admin lalu tekan Ctrl + F5.
